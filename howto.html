scraper-theparking-eu

Extract car details from www.theparking.eu website.

The input.json example: 
{
  "headless": false,
  "userAgent": "Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/129.0.0.0 Safari/537.36",
  "search_term": "VW Caddy",
  "countries": ["Germany", "France", "Belgium"],
  "from_search_page": 1,
  "max_search_pages": 100
}

Explanation of the input.json fields:

headless

Determine if the browser will be opened in the headless or non-headles (visual) mode.
Values: old, new, false
In most cases use headless: false

user_agent

The user agent which scraper will use.
Example: "Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/129.0.0.0 Safari/537.36"

search_term

It's the search term which scraper enter in the theparking.eu search field.
If it's minus (-) then all car makes will be listed.
Audi, Acura, Alfa Romeo, Bentley, Supercar, BMW, Bugatti, BYD, Chevrolet, Chrysler, Cadillac, Dodge, Fiat, Ford, GMC, Honda, Hyundai, Hummer, Isuzu, Jaguar, KIA, Lamborghini, Land Rover, Lexus, Mazda, Mini, Mercedes Benz, Mitsubishi, Nissan, Porsche, RAM, Subaru, Toyota, Tesla, Volvo

countries

It's an array of countries to filter search results. For example if you want to fetch cars only from Germany and france use "countries": ["Germany", "France"].
In case that you want to fetch cars from all countries set this array empty: "countries": []

sort_by

This parameter will determine how results will be sorted before scraper starts to extract data.
Use one of this values: date, prix_croissant, prix_decroissant, km, km_decroissant, millesime_croissant, millesime_decroissant, distance
The defult value is distance.
date - First listing date
prix_croissant - Price (ascending)
prix_decroissant - Price (descending)
km - Kilometer
km_decroissant - Kilometer (descending)
millesime_croissant - Model year (ascending)
millesime_decroissant - Model year (descending)
distance - Distance

from_search_page

Sometimes, you may need to start scraping from a specific search results page. For example, if you define "from_search_page": 5, the scraper will begin directly from search results page 5 when initiated. This approach can significantly speed up the scraping process.

max_search_pages

This parameter defines how many search pages will be scraped.
For example if you define: 
"from_search_page": 1, "max_search_pages": 100
the scraper will extract cars from page 5,6,7 ... 105
